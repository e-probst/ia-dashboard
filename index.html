⚙️ Configurar Automação (Google Apps Script)
Como configurar
1Abra script.google.com → Novo projeto
2Cole o código do arquivo CronodashMailer.gs (baixado junto com o dashboard)
3Clique em Implantar → Nova implantação → Aplicativo da Web
— Executar como: Eu (minha conta)
— Quem pode acessar: Qualquer pessoa (obrigatório para o dashboard enviar dados)
— Clique em Implantar e autorize as permissões
4Copie a URL do Web App e cole abaixo
5No editor do Apps Script, execute createDailyTrigger() UMA VEZ para agendar o envio diário às 7h
6Para testar, execute a função sendTestEmail() no editor — ela envia um e-mail de verificação
💡 Nota: O teste de conexão pode mostrar aviso de CORS ao rodar localmente (arquivo HTML) — isso é normal e não impede o funcionamento. Use o botão ↗ Abrir URL para confirmar que o Apps Script está ativo.

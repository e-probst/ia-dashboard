# Cronograma Mensal

https://e-probst.github.io/Cronograma-Mensal/

Dashboard desenvolvido para gestão e envio de cronogramas mensais com automação.

## 🚀 Funcionalidades
- Visualização de cronograma
- Automação de envio (Apps Script)
- Interface simples em HTML

## 📁 Arquivos
- Cronograma Mensal.html
- CronodashMailer.gs

## 🛠️ Tecnologias
- HTML
- Google Apps Script

## 📌 Objetivo
Facilitar o controle e envio de atividades mensais de forma automatizada.
